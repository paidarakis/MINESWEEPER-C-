### Προαπαιτούμενα
-
Windows υπολογιστής
- C++ Compiler (Visual Studio)
- Τερματικό (Command Prompt)

Αν έχεις g++, άνοιξε το Command Prompt στο φάκελο και εκτέλεσε:```bash
